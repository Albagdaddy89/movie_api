Movie API Documentation

Welcome to the Movie API! This API provides access to a vast collection of movies, allowing users to retrieve movie details, genres, directors' information, and much more. It's designed for developers to integrate movie information into their applications seamlessly.

Request URL Response
GET /movies Returns a list of all movies
GET /movies/:title Returns a single movie by title
GET /movies/genre/:genreName Returns movies by genre
GET /movies/director/:directorName Returns information about a director
POST /users Registers a new user
PUT /users/:id Updates user information
POST /users/:id/movies/:movieTitle Adds a movie to user's favorites
DELETE /users/:id/movies/:movieTitle Removes a movie from user's favorites
DELETE /users/:id Deletes a user
